Guessing Game V2 Tech Docs

This project will play a guessing game where the user will enter a range for a random number to be generated. The user will then try to guess the random number.

Getting Started

To get started, run the project in Visual Studio. A form will appear. Enter numbers for upper and lower bound and press "Start Game". A random number will be generated between the range. If an invalid range was given, the default values of 1 and 10 are used. You will then be prompted guess the random number. If the guess is incorrect, you will be given hints as to whether your guess is higher or lower than the random number. If the guess is correct, the game is finished.
